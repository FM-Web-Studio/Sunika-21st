# 📸 Drop your images here

Any image you put in **this folder** automatically shows up in the
gallery on the website — you don't need to touch any code.

**Supported file types:** `.jpg` `.jpeg` `.png` `.webp` `.avif` `.gif` `.svg`

## How to add photos
1. Copy your image files into this folder (`src/gallery/`).
2. That's it. Run `npm run dev` (or rebuild) and they appear in the
   "Gallery" section, sorted by file name.

## Tips
- Name files so they sort the way you want, e.g. `01-hero.jpg`, `02-paint.jpg`.
- The placeholder paintings (`placeholder-*.svg`) are just samples so the
  gallery isn't empty — delete them once you add your own photos.
- Big photos are fine; Vite optimizes them at build time.
